# EUTIT / EUT Interactive Tree
This project is my idea of a better way to visualize Everything Upgrade Tree's upgrades by putting them on a flat plane instead of a 3D baseplate.

My goals for this projetc are:
* To add a search feature
* To implement every upgrade
* To make an editor (because editing the JSON manually for each upgrade is tedious)

For the editor, I will most likely make it in GameMaker as it is an easier program to work with and I am most familiar with its language.
